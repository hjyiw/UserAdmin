## POST 登录

POST /auth/login

### 响应值

#### 位置

Header

#### 内容

`Authorization`: Bearer(空格) + `Jwt token`

`Set-Cookie`: `token`=`{refresh token}值` ...

> Body 请求参数

```json
{
  "username": "admin",
  "password": "test"
}
```

### 请求参数

|名称|位置|类型|必选|中文名|说明|
|---|---|---|---|---|---|
|body|body|object| 否 ||none|
|» username|body|string| 是 | 用户名|none|
|» password|body|string| 是 | 密码明文|none|

> 返回示例

> 200 Response

```json
{
  "code": 200,
  "msg": "登录成功",
  "data": {
    "token": "admin-token-1234567890",
    "userId": 1
  }
}
```

> 401 Response

```json
{
  "code": 0,
  "message": "string",
  "data": null
}
```
